## Import

```js
import {
    Section,
    SectionBody,
    SectionDivider,
    SectionHeader,
    SectionTitle,
} from 'medigate-ui/src';
```
- **Section:** wrapper
- **SectionHeader:** 헤더
- **SectionTitle:** 제목 (click tracking 기능에 사용되므로 필수) <br/>
시각적으로 필요하지않으면 SectionTitle 컴포넌트 안에 `<VisuallyHidden>제목</VisuallyHidden>`
- **SectionReadMoreButton:** 더보기 버튼
- **SectionBody:** 바디
- **SectionDivider:** 디바이더



## Variants

`basic`, `container`, `unstyled`

```jsx
<Section>
    <SectionHeader>
        <SectionTitle>제목</SectionTitle>
    </SectionHeader>
    <SectionBody>
        콘텐츠
    </SectionBody>
    <SectionDivider />
</Section>
```

## Sizes

Chakra UI provides 3 Card sizes. Use the `size` prop to change the size and set
the value to `sm`, `md`, or `lg`.

```jsx
<Stack spacing='4'>
  {['sm', 'md', 'lg'].map((size) => (
    <Card key={size} size={size}>
      <CardHeader>
        <Heading size='md'> {size}</Heading>
      </CardHeader>
      <CardBody>
        <Text>size = {size}</Text>
      </CardBody>
    </Card>
  ))}
</Stack>
```
